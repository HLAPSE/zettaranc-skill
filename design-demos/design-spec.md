# zettaranc-skill 推广页设计 Spec

## 产品定位
zettaranc（万千）是一个投资思维框架 Skill，可运行在 Claude Code、Cursor 等 AI agent 上。基于前阳光私募冠军基金经理、B站百大UP主的投资方法论，提供 60+ 技术指标、30+ 战法识别、策略回测等功能。

## 目标受众
已经在使用 AI agent（Claude Code、Cursor、Codex 等）的开发者和技术用户，对股票投资感兴趣。

## 核心信息
1. **Z 哥的投资思维框架** - 不是普通的炒股工具，是经过 200 万字深度蒸馏的投资方法论
2. **专业级功能** - 60+ 技术指标、30+ 战法识别、策略回测、智能选股
3. **无缝集成** - 直接在你的 AI agent 中使用，CLI 工具 + JSON 输出
4. **开源免费** - MIT 协议，完全开源

## 页面板块
1. **Hero 区域** - 品牌展示 + 核心价值主张 + 安装按钮
2. **功能介绍** - 六大核心能力卡片展示
3. **使用演示** - CLI 命令示例 + 效果展示
4. **安装教程** - 三步快速开始
5. **用户评价** - 社区反馈（可放 GitHub stars、用户感言）
6. **底部 CTA** - 再次强调安装 + GitHub 链接

## 情感基调
- **专业但不冰冷** - 体现投资领域的专业性，同时有 Z 哥的个人温度
- **务实不浮夸** - 「利润是市场给的，都是概率的事儿，谁也别吹牛逼」
- **技术范儿** - 面向开发者，代码示例要真实可运行

## 视觉约束
- 主色：深蓝 #1a1a2e + 金色 #e6b800（投资/金融调性）
- 辅助色：深灰 #16213e、白色
- 字体：中文用思源黑体/Noto Sans SC，英文用 Inter/JetBrains Mono（代码）
- 暗色为主，体现专业感
- 代码块要有语法高亮
- Logo 已有：assets/logo.svg

## 输出格式
单文件 HTML，1440×900 viewport，响应式适配
