Quick Start

Up and running in 3 steps

Works with Claude Code, Cursor, and any AI agent that supports skills. Zero config to start.

1

Install the Skill

One command to install. Supports npm and pip.

$ pip install zettaranc-skill
2

Configure Agent

Add the skill to your AI agent's configuration.

$ zettaranc init --agent claude
3

Start Asking

Ask anything in natural language. Z handles the rest.
